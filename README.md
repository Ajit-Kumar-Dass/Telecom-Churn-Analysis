# 📊 Telecom Churn Analysis (Power BI)

## 📌 Overview

Churn is a major concern in the telecom industry—losing customers directly impacts revenue. In this project, I explored a telecom dataset to understand why customers leave and what patterns can help predict churn.

Using Power BI, I built an interactive dashboard that highlights key trends and makes the data easy to explore from a business perspective.

---

## 🎯 What I Wanted to Solve

* Understand which customers are more likely to churn
* Identify patterns behind customer behavior
* Find factors that influence churn the most
* Present everything in a clear, interactive dashboard

---

## 📁 What’s in This Repo

* `Telecom Churn Analysis.pbix` → Main Power BI file
* Dataset (if included)
* Any additional supporting files

---

## 🛠️ Tools Used

* Power BI
* DAX (for calculations and measures)
* Power Query (for data cleaning and transformation)
* Excel / CSV dataset

---

## 📊 Dashboard Highlights

### Key Metrics

* Total Customers
* Churned Customers
* Churn Rate (%)

### Insights Covered

* Churn across different contract types
* Impact of tenure on churn
* Payment method analysis
* Service-based churn trends

### Filters Available

* Gender
* Senior Citizen
* Contract Type
* Payment Method

---

## 🔍 What I Found

* Customers on **month-to-month contracts** tend to leave more often
* **Short-tenure customers** are at higher risk of churn
* Some **payment methods** are linked with higher churn
* Customers using additional services (like tech support) are less likely to churn

---

## 💡 Why This Matters

Understanding churn helps businesses take action before customers leave. With these insights, companies can:

* Improve retention strategies
* Target high-risk customers
* Make better business decisions based on data
